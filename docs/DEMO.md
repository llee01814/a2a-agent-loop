# Demo

The v0.2 demo video is an actual dashboard walkthrough, not a slide deck. It shows the local UI while a demo council run is already recorded in the dashboard state:

1. dashboard status / event metrics
2. live branch graph for council → implementer → review
3. audit timeline with council votes and handoffs
4. human comment injection through the UI
5. stop-control path and preserved run history

Video artifact in this repository:

```text
demo/videos/a2a-agent-loop-v0.2-demo.mp4
```

Local run:

```bash
npm run demo:dashboard
```
